# WA Chat Reader by VH 📱⚖
Este projeto permite a remontagem de conversas exportadas do WhatsApp (iOS/iPhone). Originalmente, o WhatsApp exporta um arquivo `.txt` bagunçado e mídias separadas. Esta ferramenta organiza tudo em uma interface HTML intuitiva, semelhante ao layout do próprio aplicativo.

## Funcionalidades
- **Organização Cronológica:** Une texto e mídias no fluxo correto da conversa.
- **Suporte Multimídia:** Visualização direta de fotos, vídeos e player de áudio.
- **Interface Moderna:** Desenvolvido em Python com CustomTkinter.

## Como usar?
1 - **Prepare os arquivos:** Baixe o arquivo compactado da conversa exportada no seu computador e extraia-o completamente.
2 - **Execute o App:** Abra o `WA Chat Reader`.
3 - **Selecione a Pasta:** Clique no botão **"Procurar Pasta"** e selecione a pasta que contém o arquivo `_chat.txt` e as mídias.
4 - **Processe:** Clique em **"Processar"**. O programa gerará um arquivo `.html` na mesma pasta.
5 - **Visualize:** Abra o arquivo `.html` gerado em qualquer navegador (Chrome, Edge, etc).
